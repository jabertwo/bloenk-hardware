Unter Windows 8 kann der "libusb-win32" Treiber mit Hilfe von Zadig installiert werden.
Das deaktivieren der Treibersignatur Überprüfung ist hierfür nicht notwendig.

Anleitung:
	Blönk einstecken
	Zadig starten
	"Bloenk" in dem Dropdown Feld auswählen
	In dem Feld rechts neben dem "Driver" Feld "libusb-win32" auswählen
	Auf "Install Driver" klicken
